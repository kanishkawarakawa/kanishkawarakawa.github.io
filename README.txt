KANISHKA WARAKAWA - EXECUTIVE WEB PORTFOLIO

FILES
- index.html: Main website
- style.css: Responsive design and print styles
- script.js: Scroll reveal animation and current-year footer
- assets/favicon.svg: Website icon
- Kanishka_Warakawa_Executive_CV.pdf: Downloadable CV

HOW TO PREVIEW
Open index.html in a browser.

HOW TO PUBLISH
Upload every file and the assets folder to public_html, a subdomain folder, Netlify, GitHub Pages, Cloudflare Pages or another static host.

IMPORTANT
Keep index.html, style.css, script.js and the PDF in the same folder.

RECOMMENDED DOMAIN
Use a short professional domain based on the name or agency brand.
